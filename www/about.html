Our purpose is to provide an easy game recommendation service to users by utilising data science and vectorisation.

Basically, we have a dataset of tens of millions of lines for steam user profiles. We create a profile for you based on the games you've entered and (if applicable) the hours, and we run similarity search with our vectorisation algorithm to find the most similar user profiles to you and offer recommendations based on them.

We are committed to open-source code and have respect for user privacy, so we do not steal any of your data. All our source-code is accessible in our Github repository

Fret not, user. This website is 100% free for all features.
